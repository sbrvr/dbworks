# Databricks Unity Catalog Apache Iceberg REST Catalog API - Java Example
The following example shows how to connect to Databricks using Unit Catalog Apache Iceberg REST Catalog API using Java.

Prerequisites -
- Java 17
- Databricks External Access to Unity Catalog Enabled - More details [here](https://docs.databricks.com/aws/en/external-access/admin)

Set the following in the IcebergRestAPI.java file
- Databricks Workspace URL
- Service Principal PAT Token
- UC Catalog Name
- UC Schema Name
- UC Table Name
